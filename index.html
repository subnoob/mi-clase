Actividad de CSS

¿Qué es CSS?


CSS (Cascading Style Sheets) es un lenguaje de estilo utilizado para describir la presentación de documentos HTML y XML. CSS permite definir el diseño visual, la disposición y el formato de los elementos en una página web.

¿Cuál es la función principal de CSS?

La función principal de CSS es separar el contenido de un documento (escrito en HTML) de su presentación visual. Esto permite a los desarrolladores controlar el aspecto de la web, incluyendo colores, fuentes, espaciado, diseño y más, de manera efectiva.

¿Cómo se puede incluir CSS en un documento HTML?

Se utiliza dentro de una etiqueta style en la sección head del documento. No requiere atributos adicionales. Se vincula a un archivo CSS externo utilizando la etiqueta link en el head . Los atributos utilizados son rel y href .:

¿Qué son los selectores en CSS?

Son patrones que se utilizan para seleccionar elementos HTML específicos a los que se les aplicarán estilos. Permiten apuntar a un elemento o un grupo de elementos para cambiar su apariencia.

¿Qué son las propiedades y valores en CSS?

Las propiedades en CSS son las características que se pueden aplicar a los elementos HTML. Por ejemplo, color, font-size, y margin son propiedades. Cada propiedad tiene un valor asociado que determina cómo se aplicará la propiedad. Por ejemplo, en color: red;, color es la propiedad y red es el valor.

¿Cómo se aplica el modelo de caja en CSS?

El modelo de caja en CSS describe cómo se estructuran los elementos en la web. Cada elemento se representa como una caja rectangular que consta de: Contenido: el texto o imágenes. Relleno (padding): espacio entre el contenido y el borde. Borde (border): línea que rodea el relleno. Margen (margin): espacio fuera del borde que separa el elemento de otros elementos.

¿Qué es el "cascading" en CSS?

El "cascading" se refiere a cómo se aplican las reglas de estilo en CSS. Cuando hay múltiples estilos aplicados a un elemento, el navegador determina cuál aplicar basándose en: La especificidad (qué tan específico es el selector). El orden en que se declaran las reglas (la última regla que se aplica se usa). La importancia (las reglas marcadas como !important tienen prioridad).

¿Qué son las media queries?

Las media queries son una técnica en CSS que permite aplicar estilos específicos según las características del dispositivo, como el ancho de la pantalla. Esto es fundamental para el diseño responsivo.

¿Cuáles son algunos estilos comunes que se pueden aplicar?

Algunos estilos comunes en CSS incluyen: Color y fondo: color, background-color, background-image. Tipografía: font-family, font-size, font-weight, line-height. Espaciado: margin, padding. Posicionamiento: position, top, left, right, bottom. Bordes: border, border-radius.

¿Cómo se puede usar CSS para crear un diseño responsivo?

Para crear un diseño responsivo, puedes utilizar: Media queries: como se mencionó antes, para aplicar diferentes estilos según el tamaño de la pantalla. Unidades flexibles: usar %, em, rem, o vh/vw en lugar de píxeles para que los elementos se adapten mejor a diferentes tamaños de pantalla. Flexbox y Grid: estas son herramientas CSS que facilitan la creación de diseños fluidos y responsivos. Permiten alinear y distribuir espacio entre los elementos de manera más eficiente.
